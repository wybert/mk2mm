# mk2mm
Markdown to mind map. Generating freeplane/freemind/baiduNaotu mind map(`.mm`,`.km`) from markdown file. 

## How to use
Run 
```python
import md2mm
md2mm.mk2km("test.md", "test.km")
md2mm.mk2mm("test.md", "test.mm")
```
